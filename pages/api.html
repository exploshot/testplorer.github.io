API

Supported APIs

  • hashrate – current estimated network hashrate
  • height – current height (incl. genesis block)
  • reward – current block reward
  • supply – total available supply
  • lefttomine – unlocked QWC Supply
  • supplydeci – total available supply with decimals
  • supplyatomic – total available supply in atomic units
  • difficulty – next difficulty
  • transactionscount – transactionscount
  • checkpoint – checkpoint
  • checkpoint (csv) – checkpoint (CSV Export)
  • Blockchain size – Blockchain size
